jQuery AccessKey

a jQuery plugin to manage accesskeys on a document model

These pages show and demonstrate the use and functionality of the jQuery AccessKey plugin you just downloaded, based on the jQuery javascript library.

You've downloaded the whole plugin's package ; to test it in your system, please see the examples pages ; to learn how to use it and the required files, please see the usage page ; finally, a complete documentation is available in the doc page.

If you encounter errors to view these pages, please see the demo note.
